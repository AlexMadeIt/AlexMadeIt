Process

A breakdown of how this site was built from planning to final design and refinement.

01 — Planning

I started by planning the structure of the website and deciding what each section would include. I focused on keeping the layout simple and easy to navigate.

02 — Structure (HTML)

I built the foundation of the website using HTML. I organized each section clearly so the content would stay structured and readable.

03 — Styling (CSS)

I used CSS to style the website and improve the layout. This included spacing, alignment, and overall visual design.

04 — Refinement & Improvement

I reviewed the site and made adjustments to spacing and layout. This helped improve consistency and overall readability.

BUILD SNAPSHOT

This site is constantly evolving. Each section is designed to reflect progression in both structure and visual design.
